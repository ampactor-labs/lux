FONTLOG for Inka Mono
=====================

Basic Font Information
----------------------
Inka Mono is a programming font derived from JetBrains Mono, purpose-built
for the Inka programming language. It adds 17 custom ligature and replacement
glyphs designed in a geometric, angular style inspired by Incan stepped motifs.

The five pipe verb glyphs (|>, <|, ><, ~>, <~) are the visual signature of
Inka's topological architecture. The >< "bowtie" parallel compose glyph is
unique to this font — no other programming font has it.

The ? (hole) glyph is replaced with a hollow octagonal socket, visualizing
Primitive #8 (productive under error) as an invitation to fill the gap.

All glyph designs use angular/geometric forms echoing the Incan chakana
(stepped cross) motif, resonating with the language's eight-primitive kernel.
